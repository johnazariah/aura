## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| New Workflow | Ctrl+Shift+W |
| Open Chat | Ctrl+Shift+A |
| Execute Agent | Ctrl+Shift+E |
| Index Workspace | Ctrl+Alt+I |
| Show Help | Ctrl+Shift+/ |

## Command Palette
