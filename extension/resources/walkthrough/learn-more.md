# Learn More

## Documentation

Full documentation is available online and in the installation folder:

- **Use Cases** - Practical examples for common tasks
- **Cheat Sheet** - Quick reference for commands
- **Stories** - Deep dive on multi-step development
- **Chat** - Advanced code-aware chat features

Run the command **Aura: Open Documentation** to access the docs.

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| New Story | Ctrl+Shift+W |
| Open Chat | Ctrl+Shift+A |
| Execute Agent | Ctrl+Shift+E |
| Index Workspace | Ctrl+Alt+I |
| Show Help | Ctrl+Shift+/ |

## Command Palette

Press **Ctrl+Shift+P** and type "Aura" to see all commands.
