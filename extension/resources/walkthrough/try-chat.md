# Code-Aware Chat

Chat with your codebase using natural language.

## What Makes It Special

Unlike generic AI chat, Aura:
- **Searches your indexed code** for relevant context
- **Understands relationships** between components
- **Cites specific files and lines**

## Example Questions

```
How does authentication work in this project?
```

```
Where is the payment processing implemented?
```

```
What does the OrderService class do?
```

```
Show me how logging is done in this project.
```

## Tips for Better Answers

- **Be specific** - Include class/function names if known
- **Ask follow-ups** - Drill into details
- **Reference files** - "In UserController.cs, what does..."
