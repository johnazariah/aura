# Create Your First Story

Storys automate multi-step development tasks.

## How It Works

1. **Describe** what you want to build
2. **Aura analyzes** your codebase for context
3. **Aura plans** step-by-step implementation
4. **You review** and approve each step
5. **Aura executes** the approved steps

## Good Story Descriptions

✅ **Specific with context:**
```
Create a REST endpoint POST /api/products 
following the same patterns as OrderController
```

✅ **Clear requirements:**
```
Add unit tests for OrderService covering 
all public methods using xUnit and Moq
```

❌ **Too vague:**
```
Make the code better
```

## Keyboard Shortcut

**Ctrl+Shift+W** (Cmd+Shift+W on Mac) - Create new Story
