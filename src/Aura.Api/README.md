# BrightSword Project

## Project Overview
BrightSword is an open-source project designed to provide a robust framework for building scalable applications. It includes multiple modules such as Crucible, Feber, Squid, and SwissKnife.

## Architecture Summary
The architecture of BrightSword is modular, with each module serving specific purposes. Detailed architecture can be found in the [architecture.md](artifacts/docs/brightsword/architecture.md) document.

## Build Instructions
To build the project, follow these steps:
1. Clone the repository.
2. Navigate to the root directory of the project.
3. Run the following command:
   ```bash
dotnet build
```
For more detailed build instructions, refer to the [build.md](artifacts/docs/brightsword/build.md) document.

## Usage Examples
Here are some usage examples to help you get started with BrightSword:
- **BrightSword.Feber.SamplesApp**: A sample application demonstrating how to use BrightSword modules. To run it, execute the following command:
  ```bash
dotnet run --project BrightSword.Feber.SamplesApp/BrightSword.Feber.SamplesApp.csproj
```
For more usage examples, refer to the [functionbuilder.md](brightsword.feber/docs/functionbuilder.md) document.
