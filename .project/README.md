## Quick Start for Contributors

1. **New feature?** → Create a spec in `spec/`
2. **Technical decision?** → Create an ADR in `adr/`
3. **Implementation details?** → Create a plan in `plan/`
4. **Bug/enhancement?** → Create a task in `tasks/`
5. **Progress update?** → Add to `progress/YYYY-MM-DD.md`

## Naming Conventions

- **Files**: lowercase kebab-case (`my-document.md`)
- **Numbered docs**: `NNN-title.md` (e.g., `001-local-first.md`)
- **Dates**: ISO 8601 (`2025-12-12.md`)

See [ORGANIZATION.md](ORGANIZATION.md) for full conventions.
